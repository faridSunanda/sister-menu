Tahun Pelaksanaan

2024

Status

Selesai

Sumber Dana

Kemenristekdikti

Abstrak / Deskripsi

Penelitian ini bertujuan untuk mengembangkan sistem informasi akademik berbasis web yang memanfaatkan teknologi cloud computing untuk meningkatkan efisiensi dan aksesibilitas layanan akademik. Sistem ini dirancang untuk mendukung kegiatan akademik seperti pendaftaran mata kuliah, pengelolaan nilai, dan monitoring kehadiran mahasiswa secara real-time.
